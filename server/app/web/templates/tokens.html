📖 사용 가이드 (수동)

위 "바로 시작하기" 블록이 가장 쉬운 길. 직접 명령을 쓰고 싶다면 아래 참고.

1) skill (CLI) 설치

pip install --user --upgrade {{ wheel_url }}

2) 로그인 (토큰을 keyring 에 저장)

hybe-reports login --base-url {{ api_base_url }} --token <발급한 토큰>

3) 자주 쓰는 명령

hybe-reports deploy ./report-dir --visibility internal --title "Q4 분석"
hybe-reports list --mine
hybe-reports info q4-revenue
hybe-reports share q4-revenue --add sarah@hybecorp.com
hybe-reports redeploy q4-revenue ./report-dir
hybe-reports delete q4-revenue

4) Claude Desktop MCP 설정

Claude Desktop 설정 파일 (보통 ~/Library/Application Support/Claude/claude_desktop_config.json 또는 %APPDATA%\Claude\claude_desktop_config.json) 의 mcpServers 에 아래 항목 추가: 

{
  "mcpServers": {
    "hybe-reports": {
      "url": "{{ mcp_url }}",
      "headers": {
        "Authorization": "Bearer <발급한 토큰>"
      }
    }
  }
}

설정 후 Claude Desktop 재시작 → 다른 Claude 인스턴스가 같은 토큰으로 리포트 검색/조회 가능.
